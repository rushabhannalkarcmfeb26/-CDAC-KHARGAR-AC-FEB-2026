# Random Quote Generator 💬

## API Used
ZenQuotes API (https://zenquotes.io)

## Features
- Fetch random quotes
- Async/await API call
- Loading indicator
- Error handling
- Button click interaction
- Dynamic DOM updates

## How to Run
1. Open index.html in browser
2. Click "Get Quote"

## Tech Used
- HTML
- Bootstrap
- JavaScript (Fetch API, async/await)
